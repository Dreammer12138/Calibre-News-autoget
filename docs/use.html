使用

使用

后台运行

我们可以借助screen将脚本放在后台运行

$ screen -S CalibreNews

$ sudo ./news-get

Ctrl A + D退出screen即可

将其作为一个系统服务运行

后台运行的方法比较简单，但是每次开机或重启都得手动启动，当然有能力的用户可以自行编写开机启动的脚本

但是要想让它每次开机或重启都能自动开启，我们可以写一个系统服务来启动它

# 新建一个服务文件
$ sudo vim /etc/systemd/system/calibre-news-autoget.service

新建好服务文件以后，将一下代码复制进去

[Unit]
Description=Calibre-News-autoget

[Service]
Type=simple
User=root
ExecStart=/usr/bin/python3 /<脚本的路径>/news-get

[Install]
WantedBy=multi-user.target

然后执行命令启动服务

$ sudo systemctl enable calibre-news-autoget.service
$ sudo systemctl start calibre-news-autoget

这样就可以启动服务自动抓取新闻了
