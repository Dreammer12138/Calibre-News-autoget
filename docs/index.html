简述

Calibre是Linux平台上很好用的一款电子书管理软件，但是其过大的资源占用劝退了不少轻量Linux用户。

u1s1，Calibre主要有用的部分也就是书库的数据库，也就是书库路径下的metadata.md数据文件。同时Calibre同时也提供了ebook-convert和calibredb等简单好用的工具，可以支持我们进行二次软件设计。

本脚本主要是使用ebook-convert来定时更新新闻News，适合关注实事或者有意学习外语的用户。

搭配我前段时间更新的服务器段搭建远程书架的教程，可以在服务器端构建一个自动抓取新闻的云端书架系统。

脚本依赖

  • Python3
  • Calibre for Linux

本脚本是针对Linux服务器用户所设计的，桌面用户直接安装Calibre即可
